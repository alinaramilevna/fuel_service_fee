# SDS Fuel

Проект посвящён анализу поведения клиентов сети автозаправочных станций и исследованию факторов, влияющих на удержание пользователей и потребление топлива.

В рамках проекта проведён полный цикл анализа данных: от очистки и исследовательского анализа до построения математической модели оценки удержания клиентов.

---

## Цель проекта

Исследовать поведение клиентов АЗС, определить ключевые факторы, влияющие на повторные покупки топлива, и построить метрику, позволяющую оценивать вероятность удержания клиента.

---

## Основные задачи

- очистка и предобработка данных;
- исследовательский анализ (EDA);
- анализ клиентского поведения;
- построение метрики **Retention 4 Weeks**;
- исследование влияния различных признаков на удержание клиентов;
- подготовка данных для дальнейшего моделирования.

---

## Построение метрики Retention

Одной из основных задач проекта являлось построение пользовательской метрики **Retention 4 Weeks**.

Для каждого клиента определялось, совершал ли он повторную покупку топлива в течение четырёх недель после рассматриваемой транзакции.

Полученная метрика использовалась для дальнейшего анализа факторов удержания пользователей.
