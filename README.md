# FaceGuard

FaceGuard — настольное приложение на **Python + PyQt5** для распознавания лиц в видеопотоке с использованием технологий компьютерного зрения.

Проект демонстрирует применение OpenCV и библиотеки `face_recognition` для обнаружения и идентификации людей на видео, а также сохранения результатов анализа в базе данных SQLite.

---

## Возможности

- Анализ видеофайлов.
- Распознавание заранее зарегистрированных лиц.
- Сохранение результатов распознавания в SQLite.
- Просмотр сохранённых данных через графический интерфейс.
- Авторизация пользователя с хранением паролей в базе данных.
- Удобный интерфейс, разработанный на PyQt5.

## Добавление новых лиц

Для распознавания собственных изображений:

1. Поместите фотографии в папку

```
data/faces/
```

2. Загрузите изображение в `tools/cv2_functions.py`

```python
your_image = face_recognition.load_image_file(
    "data/faces/your_photo.jpg"
)
your_encoding = face_recognition.face_encodings(your_image)[0]
```

3. Добавьте кодировку в список известных лиц

```python
known_face_encodings = [
    your_encoding
]

known_face_names = [
    "Your Name"
]
```

После этого приложение сможет распознавать нового пользователя.

## Как работает приложение

1. Пользователь запускает программу.
2. Выбирает видео для анализа.
3. Программа последовательно обрабатывает кадры.
4. На каждом кадре выполняется:
   - обнаружение лица;
   - вычисление эмбеддинга;
   - сравнение с базой известных лиц;
   - сохранение результата.
5. Пользователь может просмотреть историю распознавания через интерфейс приложения.
